How do I play audio?

Click any track in the playlist on the left. Use the play/pause and seek controls in the player area.

How can I download tracks?

Individual track files can be linked or downloaded directly if available. Contact via the contact page for hi-res files.

Where are the source files hosted?

The site is a static GitHub Pages site. Track and video files are stored in /assets/.
